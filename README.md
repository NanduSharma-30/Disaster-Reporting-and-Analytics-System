# 🌍 Disaster Reporting and Analytics System using MySQL

## 📌 Project Overview

The Disaster Reporting and Analytics System is an end-to-end SQL project developed using MySQL to analyze disaster-related data and generate meaningful business insights.

This project simulates a real-world disaster management database and answers important business questions related to disasters, hospitals, shelters, rescue teams, victims, and donations using SQL.

The project demonstrates SQL concepts ranging from basic querying to advanced analytics, including joins, subqueries, Common Table Expressions (CTEs), and window functions.
